Documentation

Goal

Create a game using JavaScript without using canvas

Summary

I wanted to make a different version of a classic game, Snake. The problem I had with Snake was that it doesn't get faster it just gets more tedius.
Using a grid and animation frames I was able to re create snake and add my own changes to it. A level system that increases the speed of the snake and resets the snake every level.
I worked on the recreation of snake to get the base of the game working right first.
Next on the reset function which is where I had a lot of trouble because for a while the reset would cause a collision with itself for no real reason.
After that I fixed the reset and had a different problem where the reset removed all self collisons and after a while of debugging I found out it was caused by a missing !
Next I worked on sound and adding effects for pickup death and levelup this was very easy.
I wanted to add a song like in an arcade but was unable to find a free to use song that sounded like an arcade so I only went with the effects.
Finally, I worked on the css which took a while because of small things with a grid layout and importing font errors but eventually I figured out all the errors.
